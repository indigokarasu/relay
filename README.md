# Relay

Device gateway. Ingests telemetry, messages, and attachments from Relay.app clients. Normalizes device context. Manages permissions with audit trails.

## Repository Structure

- `ocas-relay/` - Skill package source
- `spec/` - Specification and build documentation

## Installation

This skill package is available on GitHub. To use it, clone this repository and follow the installation instructions in your skill management system.
